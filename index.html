FAQ

Direct answers to most common user questions. Please don't hesitate to ask, we're to help further.

Do my users need to install anything?
No! Your users don't need to install anything to see popups on your site. Popupular integrates directly into your sight, so your popups will be seen in any desktop or mobile browser without the need to install anything.
Can Popupular be used on Wordpress, Shopify, Wix, etc?
Yes! Popupular can be installed on any site where you can add Javascript tags to the Header or the <head> section of the source code.
Can Popupular be used on subdomains or multi-tenant web apps?
Yes! In fact you will only need to install the Javascript tag once and it will work across all subdomains, if you so choose. We also provide flexible targeting rules so you can cherry pick which (sub) domains should display which popups.
What happens when I exceed my plan limits?
It won't abruptly stop working! We'll notify you 2 times (at 90% and 95%) before your limit runs out and once again when it's been fully exceeded. After that we provide a 3-day grace period where your popups will continue to show. This helps avoid sudden disruption of service and gives you time to upgrade.
